
### 프로젝트 소개 ( 뉴맵 )

지하철 도착정보를 보여주는 어플리케이션, 사용자가 평소에 자주 찾는 노선을 즐겨찾기에 추가하여 접근성을 높히고 신속하게 도착정보를 찾을 수 있습니다.

지하철을 타면서 현재 위치를 몰라 내릴역을 놓치는 분들을 위해 현재 위치를 표출 해주거나, 자신의 내릴역이 다가오면 알림으로 알려주는 기능을 통해 사용자에게 편리함을 제공하는 어플리케이션을 목표로 하고 있습니다.
